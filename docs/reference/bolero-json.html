Json

Namespace: Bolero

Customize JSON serialization for remoting.

Nested types and modules

TypeDescription
ConstantAttribute

Declares that a union case should be de/serialized as a constant.
DateTimeFormatAttribute

Defines the format used to de/serialize a DateTime field or union case argument. The default is "o" (ISO 8601 round-trip format). Note: this cannot be used on DateTimeOffset; it always uses "o" format.
Decoder<'T>

A JSON decoder from a given type.
DecoderException

Thrown when the decoder fails to reconstruct a value from JSON.
Encoder<'T>

A JSON decoder from a given type.
EncoderException

Thrown when the encoder is given a value it cannot encode.
NameAttribute

Defines a JSON serialized name for a property or field when it differs from the F# name.
NamedUnionCasesAttribute

Declares that when de/serializing this union, its fields must be tagged by their name rather than "\(0" ... "\)n". Also determines how the cases are distinguished, instead of the default "$": .
NoDecoderException

Thrown when no decoder can be derived for a given type.
NoEncoderException

Thrown when no encoder can be derived for a given type.
Value

Represents a JSON value.
ModuleDescription
Raw

Plain JSON serialization and deserialization from and to Value.
